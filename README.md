# Ebin Biju Portfolio Website

Simple personal portfolio site for showcasing my profile, resume, and featured projects.

## What’s in this repo

- `index.html` – homepage with the intro, profile summary, tech stack, education, and contact links
- `projects.html` – project gallery with project cards, screenshots, short descriptions, skill tags, and GitHub links
- `assets/` – images and files used by the site, including project screenshots and the CV PDF

## Built with

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Font Awesome

## How to view it locally

You can open `index.html` directly in a browser, or serve the folder with any simple static server.

Example:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```
